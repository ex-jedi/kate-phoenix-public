# Perch Site Setup #

1. Setup Trello board.

2. Setup and clone Git repo.

3. Configure local, staging (& live) servers. Add details to LastPass.

4. Setup local, staging (& live) database. Add details to LastPass.

5. Buy Perch licence and enter domain details.

6. Add site files to local host.

7. Initial Perch Setup:
    **Delete Core gitignore file.**
    Activate Perch.

8. Configure Deploy.hq:
     Check gitignore file.
     Check deployignore file.
     Enter FTP details (& save to LastPass).
     Add Perch config file for staging (& live files).

9. Initial Git commit & push.

10. Configure Perch CMS on staging (& live) server:
    Create (or copy) resources folder
    (Copy local database.)
    Delete Perch setup file.

11. Configure Prepros/Gulp.

12. Add resources/content to Trello & Movable Shed.

13. Send details to client.
